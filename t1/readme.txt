学籍管理系统

    网站程序员：PHP + Mysql

5.3 功能模块
5.3.1   学生信息管理    包含信息：姓名，学号，性别，班级，出生年月，入学年份，籍

贯【省份】，备注    需要实现功能
    1). 学生信息显示
    2). 学生信息的增删改
5.3.2   科目信息管理
    包含信息：科目编号，科目名称，学分，课时，授课教师，备注    需要实现的功能
a)      科目信息显示界面
b)      科目信息增删改查
5.3.3   学生考试成绩管理    包含信息：成绩编号，学年，学生编号，科目编号，成绩，备注
    需要实现功能
a)  录入学生成绩
b)  可查询一个学生在某一学年的各科成绩
c)  可查询某科目下在一个学年中所有参加考试学生的成绩
d)  学生各科成绩可排序
5.3.4   学生成绩报表
a)      各学生各学年的总分。
b)      可按总分进行排序。
